Mildom Chat CSS Generator

1. OBSのブラウザソースに「https://www.mildom.com/ユーザID」を指定します。
2. 横幅を800px以上に指定します。（高さは任意で良い）
3. そのまま追加し、「対話」ウィンドウからブラウザソース内のミルダムをミュートにします。
4. 改めてブラウザソースのカスタムCSSに、本サイトで生成したCSSを追加します。
